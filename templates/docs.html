RoosterAPI Documentatie

RoosterAPI is een A.P.I. (application programming interface) voor het rooster van het horizon college.
Deze A.P.I. kan worden gebruikt om een GET request te sturen.
De A.P.I geeft dan een lijst met allemaal 'les' objecten terug. Daarin staat de informatie van de specifieke les.

De A.P.I. wordt gehost met heroku, op de volgende url: https://horizon-rooster.herokuapp.com/api/rooster/

Output

De output van de A.P.I. is een lijst met JSON objecten.
Deze objecten hebben allemaal dezelfde opbouw, maar andere informatie. Hieronder is een voorbeeld

	{
		"vak": "ENG",
		"docent": "WEELD0",
		"plaats": "online",
		"wanneer": {
			"dag": "Dinsdag 13-04",
			"tijd": {
				"start": "09:20",
				"eind": "10:10"
			}
		}
	}
*Het 'plaats' veld kan ook het lokaalnummer bevatten.

Input

De volgende URL wordt aangenomen:

  • https://horizon-rooster.herokuapp.com.api/rooster

Echter zijn er meer opties. Via URL queries kan er meer worden gespecificeerd. Deze queries kunnen worden verdeelt in 2 Soorten: Filters en Url kiezers.
De filters kunnen worden gebruikt om bijvoorbeel een specifiek vak te kiezen, de Url kiezers worden gebruikt om te bepalen uit welk rooster de lessen worden gehaald

De url queries kunnen natuurlijk ook worden gecombineerd, bijvoorbeeld ?week_nummer=16&vak=PORTF

Filters

De filters veranderen niet wat voor data wordt opgehaald, alleen welke data wordt teruggegeven.

  • ?vak=PORTF

    Filtert het vak. Alleen de lessen van dit vak worden laten zien.

  • ?docent=KERCK0

    Filtert de docent. Alleen de lessen van deze docent worden laten zien.

  • ?plaats=C207

    Filtert de plaats/lokaal. Alleen de lessen op deze locatie worden laten zien.

  • ?dag_van_week=2

    Filtert de dag van de week. Alleen de lessen op deze dag worden laten zien. Dit is zero-based.(maandag=0, dinsdag=1 etc.)

Url Kiezers

  • ?sector=ECO

    Je kan specificeren uit welke sector het rooster wordt opgehaald. De standaard waarde is ECO.

  • ?gebouw=HRN

    e kan specificeren uit welk gebouw het rooster wordt opgehaald. De standaard waarde is HRN(hoorn).

  • ?klas=H19AO-A

    Je kan specificeren uit welke klas het rooster wordt opgehaald. De standaard waarde is H19AO-A.
    Om te bepalen wat je hier moet invullen, kijk je op de originele site, en vul je in wat er in het 'Element' vak staat.

  • ?week_nummer=15

    Ke kan specificeren van welke week het rooster wordt opgehaald. De standaard waarde is de huidige week.
    Dit kan niet oneindig ver terug of vooruit. Meestal kan er 1 week terug en 2 weken vooruit gegaan worden, maar dit staat niet vast.
